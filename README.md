## 서비스 소개
### 배달 앱 '[배달의 민족](https://apps.apple.com/kr/app/%EB%B0%B0%EB%8B%AC%EC%9D%98%EB%AF%BC%EC%A1%B1/id378084485)' 클론코딩 프로젝트
* 스마일 아이콘을 클릭하여, 기존 아이디 / 네이버 아이디로 **로그인**해주세요.  
  ✔ 로그인이 되어 있을 시에는, 'My배민' 페이지로 이동합니다.
* 내 정보 수정 페이지에서, 이름ㆍ비밀번호ㆍ전화번호를 변경할 수 있습니다.
* 메뉴 네비게이션 아이콘을 클릭하여, 내 위치 기반 **우리 동네 코로나19 예방접종센터**를 확인할 수 있습니다.
* 주소 텍스트를 클릭하여, **원하는 배송 위치로 검색ㆍ변경**할 수 있습니다.
* 배달 탭을 클릭하여, **배달 가게 정보를 확인**, **통화**할 수 있습니다.
* 안심식당 탭을 클릭하여, **농림축산식품부 선정 안심식당** 정보를 확인할 수 있습니다.
* **리뷰**를 작성한 후, 목록을 확인ㆍ수정ㆍ삭제할 수 있습니다.
